⚛️ Remotion

Descreva o que você quer no vídeo — a IA gera o código React/Remotion completo, pronto para rodar no seu PC e exportar em MP4.

1
Instalar Node.js
Baixe e instale o Node.js LTS em nodejs.org — necessário para rodar o Remotion.
2
Criar projeto Remotion
Abra o terminal (CMD ou PowerShell) e rode:
npx create-video@latest
Escolha o template Hello World quando perguntar.
3
Substituir o arquivo gerado
Cole o código gerado aqui no arquivo indicado dentro da pasta src/ do seu projeto.
4
Visualizar e exportar
No terminal, dentro da pasta do projeto:
npm run dev # abre o studio no browser
npx remotion render # exporta o MP4
